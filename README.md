# Leopher 3.0

Project encrypts and decrypts messages through Caesar and Vigenère Cipher algorithms.

**Coded by theLionborn - Leo Hoxha**

Version 3.0 - February 2026

**Compatibility: Python 3 Standard**

## Usage

Run the script in Python:

`python3 leopher.py`

Follow the UI next through keyboard input.

## Features

- Caesar Encryption & Decryption
- Vigenère Encryption & Decryption
- Text file encryption & decryption support
  - includes importing key from file
- Multi-lingual support

### Current languages supported:

- English (Default)
- Turkish (includes 'Q', 'X', 'W')
- Albanian (includes 'W')
- Serbian (Latin)
- Russian
- Arabic
- Hebrew
